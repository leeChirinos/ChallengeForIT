# Esta es una aplicación gestor de tareas que permite gestionar tareas: crear, ver, modificar, eliminar y filtrar por estado.

## Tecnologías usadas

- *Frontend:* React + Vite
- *Backend:* Node.js + Express

## Instalación y ejecución
npm install
npm run dev
